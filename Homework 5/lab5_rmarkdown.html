plot(mtcars$hp, mtcars$mpg)

model1 = lm(mpg~hp, data=mtcars)

The regression coefficient of the intercept is 30.099, means that this is the point where the function crosses the y-axis, in this case, we would start at 30.099 mpg for 0 hp. The regression coefficient of hp is -0.06823, this means when hp increase by 1 unit, the mpg would decrease by 0.068 unit. R-squared is 0.6024, so there’s about 60% can the variance for mpg can be explained by hp.
